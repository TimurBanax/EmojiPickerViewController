# EmojiPickerViewController
A picker view controller for emoji.

## Features
- [x] Emoji picker
- [] Segmented control for jumping a emoji section
- [] Select skin tones from popup
- [] Seach bar and search results
- [] Recently used
- [] Fully accessible
